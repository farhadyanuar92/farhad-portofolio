## Overview

A personal portfolio site built to be fast, minimal, and readable. The goal was simple: present work clearly without getting in the way. Built on Next.js App Router with the Once UI component system, deployed as a static-first site.

## Technical Decisions

- **Next.js App Router** — leverages React Server Components for zero-JS-by-default pages, keeping client bundles small.
- **MDX for project content** — project and blog content lives in `.mdx` files, keeping writing separate from component logic and making it easy to update without touching the codebase.
- **Once UI** — used as the base component system for layout primitives, letting me focus on content and information architecture rather than rebuilding common UI patterns.
- **TypeScript throughout** — all configuration and content types are fully typed, catching mistakes early.

## Stack

- **Next.js, React, TypeScript**
- **Once UI component system**
- **MDX**
- **Deployed on Vercel**
